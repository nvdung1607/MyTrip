# MyTrip 🗺️

Ứng dụng Android quản lý chuyến du lịch — lên kế hoạch, theo dõi trong chuyến đi, và tổng kết sau chuyến.

## Tính năng chính

- 📋 **Lên kế hoạch**: Tạo lịch trình chi tiết theo cụm/ngày, dự kiến chi phí, import/export Excel
- 🗺️ **Trong chuyến đi**: Theo dõi lịch trình, ghi note kèm ảnh, nhắc nhở thông báo, ghi nhận chi phí thực tế
- 📊 **Sau chuyến đi**: Tổng kết hành trình, báo cáo chi phí dự kiến vs thực tế, chia đầu người

## Công nghệ

- **Platform**: Android Native (Kotlin + Jetpack Compose)
- **Database**: Room (SQLite — lưu local)
- **UI**: Material 3
- **Ngôn ngữ**: Tiếng Việt
